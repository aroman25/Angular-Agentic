# Source Skill Map (Codex Angular Skills -> Orchestrator Skill Pack)

This orchestrator skill pack is a synthesis of patterns from the local Codex Angular skills, tailored for the Planner/Developer/Validator workflow.

## Primary sources synthesized

- `angular-dev` -> general Angular implementation workflow and debugging
- `angular-app-auditor` -> audit sequencing and prioritized findings mindset
- `angular-forms` -> form modeling, validation, error UX
- `angular-testing` -> test expectations and verification flow
- `angular-accessibility` / `angular-accessibility-plus` -> keyboard/focus/ARIA checks
- `angular-routing-guards-resolvers` -> route/default-view planning
- `angular-state-reactivity` -> signals-first local state design
- `tailwindcss` -> utility-first styling and config troubleshooting
- `new-app-guidelines` -> responsive/layout/theming guardrails

## Why this is not a 1:1 copy

- The orchestrator needs agent-role prompts, deterministic checks, and template-first escalation logic.
- This pack removes generic tutorial content and keeps only reusable orchestration guidance.
- The pack is explicitly version-aware for Angular `v20` and `v21`.
